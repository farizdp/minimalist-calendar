# Minimalist Calendar

A clean, minimal calendar for personal use — highlights Arsenal FC match days and Indonesia public holidays.

## Features

- Displays current date and month prominently
- Small annotation/remark text on notable dates
- Background turns red on Arsenal FC match days
- Indonesia public holiday integration

## Data Sources

- **Indonesia holidays** — [libur.deno.dev](https://libur.deno.dev/)
- **Arsenal FC calendar** — [ecal.com Arsenal FC feed](webcal://ics.ecal.com/ecal-sub/69e78417a04db9000291d29e/Arsenal%20FC.ics)
